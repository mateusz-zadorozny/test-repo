Interactive module with buttons leading to outside pages of partners.

== Description ==

Partners may be added or edited in the \"Partners\" section. Search for partner avname to display it on the list.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
