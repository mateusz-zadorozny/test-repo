Interactive module with 5 chosen segments showing a shortcut of specific article. 

== Description ==

Clicking on a segment is leading to an article. One (most important) article is highlighted with a different segment size. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
