Interactive or non-interactive module, possibly with button. 

== Description ==

Requires good quality photo as a background. There is no need to prepare the photos in order to make menu bar visible - menu bar contains a black gradient making the tabs always visible.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
