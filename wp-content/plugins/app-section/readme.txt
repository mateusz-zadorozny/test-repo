Interactive module with information about FutureHome mobile apps. 

== Description ==

Two buttons letting user choose from App Store (for iPhones) and Google Store (other devices). Graphic elements showing the app open on two undefined devices.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
