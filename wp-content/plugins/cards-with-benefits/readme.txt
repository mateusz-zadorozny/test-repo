Interactive or non-interactive module with customizable columns grid. 

== Description ==

Possibility of 3 types of grid composition (1, 2 or 3 columns enlarged in the upper part of module). Each column may be turned into interactive element leading to a sub-page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
