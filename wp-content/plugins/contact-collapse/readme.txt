Interactive module prepared for contact details. 

== Description ==

Unfolded segment shows the category/kind of the contact person, details are unhidden after click. The box on the right may display the fluentform shortcode.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
