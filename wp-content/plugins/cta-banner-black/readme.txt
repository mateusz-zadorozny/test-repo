Interactive banner with a button leading to specific section or outside link. 

== Description ==

It is possible to insert fluentform shortcode instead of CTA button.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
