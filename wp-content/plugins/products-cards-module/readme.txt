Module for products from partners. 

== Description ==

Product card should display following information: photo in PNG format with transparent background, name of the product and the company logo.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-module` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
